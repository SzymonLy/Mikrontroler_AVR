Plik bin_gen.py jest używany do utworzenie pliku wysyłanego na układ FPGA.
Plik hex_gen.py jest używany do utworzenia pliku pamięci programu używanym przez symulator.
MAKEXCOE jest od dawna nie używane, niemniej jest alternatywą wgrywania pamięci programu, ponieważ pliki .coe zawierają opis pamięci ram odczytywany przez vivado w trakcie syntezy bloku IP.
